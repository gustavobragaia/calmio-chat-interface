Calmio — Plataforma de Apoio Emocional

    O Calmio é um aplicativo web focado em saúde emocional, oferecendo um chatbot inteligente e acolhedor, apoio guiado, ferramentas de bem-estar e interface moderna.
    O projeto foi desenvolvido em React + TypeScript com foco em usabilidade, acessibilidade e experiência do usuário.

Tecnologias utilizadas

    O projeto utiliza:

        React 18

        TypeScript

        Vite

        Tailwind CSS

        shadcn/ui

        React Router

        Lucide Icons

        Axios
